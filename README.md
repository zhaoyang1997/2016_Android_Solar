# 2016级项目实训成果展示 

## 《Solar》 - Java与移动智能设备开发

### 项目简介

    这是一款高效习惯养成的App，结合了科学的番茄工作法，有序的计划制定，任务管理等功能，将你的工作和生活中的待办任务与目标技能关联起来，让你做过的事务都成为积累技能成长的一部分。通过丰富的图表统计，简洁的事务提醒，为你直观展示你对时间的管理情况，打造一款全方位的实用性时间管理工具。
